# LiFi Morse Code AI Relay

## Overview

LiFi Morse Code AI Relay is an optical wireless communication system that uses LED light to transmit Morse code messages between devices. The received message is decoded and processed by a local AI model to generate intelligent responses.

### Features

* Li-Fi based communication
* Morse code encoding and decoding
* Arduino UNO implementation
* LCD display output
* Python serial communication
* AI-powered reply generation using Ollama
* Low-cost and scalable design

### Technologies Used

* Arduino UNO
* LDR Module
* LCD Display with I2C
* Python
* PySerial
* Ollama
* Local Large Language Model (LLM)

### Applications

* Smart classrooms
* Secure indoor communication
* IoT networks
* RF-restricted environments
* AI-assisted optical communication systems
